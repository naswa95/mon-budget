# MonBudget V2

Projet React + Vite prêt pour GitHub Pages.

## À faire avant l'envoi

1. Ajoutez votre logo sous le nom `icon.png` dans le dossier `public/`.
2. Vérifiez `vite.config.js`. Le dépôt étant `mon-budget`, la valeur est `base: "/mon-budget/"`.
3. Envoyez tout le contenu du dossier à la racine de votre dépôt GitHub.
4. Dans Settings > Pages, choisissez GitHub Actions.

## Logo

- Nom exact : `icon.png`
- Emplacement : `public/icon.png`
- Format recommandé : PNG carré 1024 × 1024 px

## Première ouverture

L'application demande le prénom, le salaire et l'objectif d'épargne annuel. Les données restent dans le navigateur via localStorage.
